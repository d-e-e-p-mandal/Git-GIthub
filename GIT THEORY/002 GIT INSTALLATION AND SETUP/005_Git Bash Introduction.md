
# Git Bash Introduction

## What is Git Bash?

Git Bash is a terminal application for Windows that provides:
- Linux-like shell
- Git commands
- Bash scripting support

It combines:
- Git tools
- Bash shell
- Unix commands

---

## Features of Git Bash

- Supports Linux commands
- Supports Git commands
- Lightweight terminal
- Easy navigation
- Supports scripting

---

## Common Git Bash Commands

### Show Current Directory

```bash
pwd
```

### List Files

```bash
ls
```

### Change Directory

```bash
cd folder-name
```

### Create Folder

```bash
mkdir project
```

### Clear Terminal

```bash
clear
```

# Initial Git Configuration

## Why Configuration is Needed?

Git uses configuration settings for:
- Author information
- Default editor
- Repository behavior
- Authentication

Git stores:
- Username
- Email
- Preferences

---

# Levels of Git Configuration

Git has three configuration levels:

| Level | Scope |
|---|---|
| System | Entire computer |
| Global | Current user |
| Local | Specific repository |

Priority order:

```text
Local > Global > System
```

- Local overrides Global.
- Global overrides System.
