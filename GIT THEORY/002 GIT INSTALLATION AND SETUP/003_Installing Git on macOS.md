# Git Installation & Configuration

# Installing Git on Windows

## Step 1: Download Git

Visit the official Git website:

https://git-scm.com

Click on:
- Download for Windows

The installer file (.exe) will start downloading.

---

## Step 2: Run the Installer

Open the downloaded file and follow installation steps.

### Important Options During Installation

### Select Components
Recommended:
- Git Bash
- Git GUI
- Windows Explorer Integration

### Choosing Default Editor
You can choose:
- VS Code
- Vim
- Notepad++
- Nano

VS Code is beginner friendly.

---

### Adjusting PATH Environment

Recommended option:
- Git from command line and third-party software

This allows Git commands from:
- CMD
- PowerShell
- VS Code Terminal

---

### HTTPS Transport Backend

Recommended:
- Use OpenSSL library

---

### Configure Line Ending Conversions

Recommended:
- Checkout Windows-style, commit Unix-style line endings

---

### Terminal Emulator

Recommended:
- Use MinTTY

---

### Final Installation

Click:
- Install
- Finish

Git is now installed.

---

# Installing Git on Linux

Different Linux distributions use different package managers.

---

## Ubuntu / Debian

Update packages:

```bash
sudo apt update
```

Install Git:

```bash
sudo apt install git
```

---

## Fedora

```bash
sudo dnf install git
```

---

## CentOS

```bash
sudo yum install git
```

---

## Arch Linux

```bash
sudo pacman -S git
```

---

# Installing Git on macOS

## Method 1: Using Homebrew

Install Git:

```bash
brew install git
```

## Install Homebrew (if not installed)

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

---

## Method 2: Using Xcode Command Line Tools

```bash
xcode-select --install
```

This installs:
- Git
- Compiler tools
- Command line utilities

---

# Verifying Git Installation

After installation, verify Git is installed properly.

## Check Git Version

```bash
git --version
```

*Example output:*

```text
git version 2.45.1
```

---

## Check Git Help

```bash
git help
```

If help opens successfully, Git installation is working correctly.
