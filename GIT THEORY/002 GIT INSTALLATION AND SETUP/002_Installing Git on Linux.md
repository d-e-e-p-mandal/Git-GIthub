
# Installing Git on Linux

Different Linux distributions use different package managers.

---

## Ubuntu / Debian

Update packages:

```bash
sudo apt update
```

Install Git:

```bash
sudo apt install git
```

---

## Fedora

```bash
sudo dnf install git
```

---

## CentOS

```bash
sudo yum install git
```

---

## Arch Linux

```bash
sudo pacman -S git
```
